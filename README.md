# fatality-auto-fat-sale
自动销售Fatality.win 的fats工具，需要一台Winserver

# 介绍
## 使用Python编写，Flaks开接口，前端（卡网）对接进行购买请求。
## 后台完全对接易支付，按照易支付旧接口进行对接，可以自行修改。接口地址、商户ID、商户秘钥请自行修改，main.py,tools.py,refund.py或其他，自己找吧。
## 我自己写了一个对接这套系统的浏览器插件，在auto_clear_tab文件夹，将他用开发者模式加载进浏览器就会自动尝试连接后端。
# 怎么使用
 - Winserver 任意版本
 - Google Chrome（我用的是，其他的不知道行不行）
 - Python 3.12.0（我的，其他版本说不定出现什么错误）
 - pytesseract 任意版本 需安装English字库，后端中调用路径在tools.py自己改
## 用pip安装好依赖库之后python main.py即可启动，同时打开浏览器，登录好FA账号，打开浏览器插件，后端显示插件已连接即可。

# 注意事项
 - 完全基于OCR识图+模拟点击，效率低下。
 - 需要保持浏览器不关闭（会自动清理tabs，不必担心内存）
 - 代码很垃圾，没什么可读性
 - 完全可以搓一个通过浏览器插件模拟DOM点击&input输入，效率更高更快。（在做了）
